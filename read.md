# HR 360 - Human Resources Management System

Welcome to **HR 360**, a comprehensive frontend web application designed for managing human resources efficiently.

## 🚀 Overview

HR 360 is a React-based application built with [Vite](https://vitejs.dev/) to provide a fast, responsive, and modern user experience. It serves as a central hub for managing employee data, attendance, reports, and more. 

## 🎯 Key Features

Based on the project structure, here are the main modules included in this dashboard:
- **Dashboard**: A central overview of key metrics and recent activities.
- **Employees**: A complete directory to manage employee profiles and details.
- **Attendance**: Tracking and managing employee attendance and working hours.
- **Applications**: Managing leave requests and other applications.
- **Reports**: Generating and viewing HR-related reports and analytics.
- **Leaderboard**: A gamified module likely tracking performance or achievements.
- **Notifications**: System alerts and updates.
- **Settings**: Configuration options for the platform.
- **Authentication**: Secure login and session management.

## 🛠️ Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS & Framer Motion (for animations)
- **Routing**: React Router DOM
- **Icons**: Lucide React
- **Data Visualization**: Recharts

## 💻 How to Run Locally

1. **Install Dependencies**: Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```
2. **Start the Development Server**:
   ```bash
   npm run dev
   ```
3. **View in Browser**: Open the local URL (usually `http://localhost:5173`) provided in your terminal.

## 📁 Project Structure

- `src/pages/` - Contains all the main pages (Dashboard, Employees, Attendance, etc.)
- `src/components/` - Reusable UI components.
- `src/context/` - Global state management (e.g., AuthContext).
- `src/mocks/` - Mock data used for frontend development and testing.
